## 1. 总览：三栏布局结构

```

│                    顶部导航栏（48px，固定）                             │

│（可折叠）   │                              │   （可折叠，三Tab）         │
│ 默认240px  │                              │   默认400px               │
│            │   ┌──────────────────────┐   │                           │

### 1.1 栏宽默认值与约束

| 区域 | 默认宽度 | 最小宽度 | 最大宽度 | 折叠后 |
|------|---------|---------|---------|--------|
| 左侧导航栏 | 240px | 180px | 360px | 0（隐藏） |
| 中间对话区 | 自适应填充 | — | — | 填充剩余空间 |
| 右侧工作区 | 400px | 280px | 600px | 0（隐藏） |

### 1.2 分隔条规格

- 左侧与中间之间：可拖拽分隔条，宽度 4px，hover 时变为 8px 并改变颜色示意可拖拽
- 中间与右侧之间：同上
- 分隔条光标：`col-resize`
- 拖拽范围详见 [第6章](#6-拖拽与折叠交互)

## 2. 顶部导航栏

### 2.1 布局规格

```
┌──────────────────────────────────────────────────────────────────┐
│ [SciAgent Logo]  │  🔍 搜索文献/对话/文件...  │  🔔  │  👤 用户 ▾ │
│                 │                             │      │           │
└──────────────────────────────────────────────────────────────────┘
```

| 区域 | 对齐 | 内容 |
|------|------|------|
| 左侧 (flex-start) | 左 | 产品 Logo（32×32 SVG）+ 名称 "SciAgent"（字体 18px，字重 600） |
| 中间 (flex-center) | 中 | 全局搜索框（宽度 320px，高度 32px，圆角 16px） |
| 右侧 (flex-end) | 右 | 通知图标（badge 显示未读数）+ 用户头像（32×32 圆形） |

### 2.2 全局搜索框

- **唤起方式**：点击搜索框 或 快捷键 `Ctrl+K`
- **唤起效果**：弹出搜索面板（Overlay + 半透明遮罩），包含：
  - 搜索输入框（自动聚焦）
  - 分类 Tab：全部 / 文献 / 对话 / 文件 / 产出物
  - 搜索结果列表（实时联想，防抖 300ms）
  - 每项结果显示：类型图标 + 标题 + 简短描述 + 时间
- **关闭方式**：点击遮罩 / `Esc` / 选择结果后自动关闭

### 2.3 通知图标

- 未读通知数 badge：红色圆点 + 白色数字，右上角偏移
- 点击弹出通知面板（Dropdown，宽度 360px，最大高度 400px，可滚动）
- 通知项：图标 + 标题 + 时间 + 已读/未读状态
- 底部「查看全部通知」链接

### 2.4 用户头像下拉菜单

点击触发 Dropdown（宽度 200px）：

| 菜单项 | 说明 |
|--------|------|
| 个人信息 | 跳转个人信息编辑页 |
| 设置 | 通用设置、快捷键、主题 |
| 帮助文档 | 打开帮助中心（右侧预览Tab） |
| 退出登录 | 清除 Token，跳转登录页 |

## 3. 左侧导航栏

### 3.1 整体结构

```
┌─────────────────────┐
│ 【固定区】           │  ← 不可折叠，始终显示
│ ┌─────────────────┐ │
│ │ [+ 新建对话]    │ │  ← 主按钮，全宽，高40px
│ └─────────────────┘ │
│ ┌─────────────────┐ │
│ │ 会话历史列表     │ │  ← 可折叠分区
│ │ · 对话1         │ │
│ │ · 对话2         │ │
│ │ · ...           │ │
│ └─────────────────┘ │
├─────────────────────┤
│ 【功能入口区】       │  ← 可折叠分区
│ ┌─────────────────┐ │
│ │ ▸ 自动任务      │ │
│ │ ▸ 工具广场      │ │
│ │ ▸ 科研全流程    │ │
│ │ ▸ 本地知识库    │ │
│ └─────────────────┘ │
└─────────────────────┘
```

### 3.2 固定区

#### 3.2.1 新建对话按钮

- **样式**：主色填充按钮，全宽，左侧 "+" 图标，文字 "新建对话"
- **点击行为**：
  1. 后端调用 `POST /conversations` 创建新会话
  2. 会话历史列表顶部插入新条目（高亮为 active）
  3. 中间对话区清空，显示欢迎引导页

#### 3.2.2 会话历史列表

- **分组标题**："最近对话"（右侧有折叠/展开箭头）
- **列表项规格**：
  - 每项高度 40px，hover 背景色变化
  - 显示：对话标题（单行截断，最大 24 字符）+ 右侧时间戳（相对时间）
  - 当前活跃对话：左侧色条高亮 + 背景色加深
- **右键菜单**（或长按）：
  - 重命名 → 弹出内联编辑框
  - 删除 → 确认弹窗后调用 `DELETE /conversations/{id}`
- **搜索会话**：列表顶部小搜索框，实时过滤列表
- **折叠行为**：点击分组标题右侧箭头折叠/展开，状态持久化到 `localStorage`

### 3.3 功能入口区

#### 3.3.1 自动任务

- **展开/折叠**：点击分组标题切换
- **子菜单项**：

| 子项 | 点击行为 |
|------|---------|
| 文献追踪 | 中间对话区开启配置引导，自然语言 + 表单双模式配置 |
| 日报生成 | 同上，配置日报模板和推送时间 |
| 科技快讯 | 同上，配置关注领域和关键词 |
| 自定义任务 | 打开 Cron 任务配置面板（类似定时任务 UI） |

- **自然语言创建支持**：对话区输入如"每天早上8点帮我查最新的量子计算论文"时，系统自动识别意图并引导配置

#### 3.3.2 工具广场

- **点击行为**：中间对话区切换为工具广场视图（SkillHub 风格）
  - 卡片网格布局展示可用 Skills
  - 搜索 + 分类筛选
  - 每个 Skill 卡片：图标 + 名称 + 简介 + 安装/卸载按钮

#### 3.3.3 科研全流程

- **展开后子项**：

| 序号 | 子项 | 点击行为 |
|------|------|---------|
| 1 | 文献检索 | 中间对话区顶部显示检索栏（多条件筛选），对话模式切换为检索模式 |
| 2 | 知识库 | 中间对话区切换为个人知识库管理视图（文件夹树 + 文献列表） |
| 3 | 常用写作指引 | 弹出写作模板选择器（综述/论文/报告/专利），选择后加载模板到对话区 |
| 4 | 仿真沙盒 | 右侧工作区自动展开并切换到「工作」Tab，初始化沙箱终端 |
| 5 | 协作空间 | 中间对话区切换为协作空间视图（空间列表 + 动态流） |
| 6 | 算法商城 | 中间对话区切换为算法商城视图（类似工具广场） |

#### 3.3.4 本地知识库

- **子项**：我的文献 / 我的笔记 / 实验记录
- **点击行为**：中间对话区切换为对应管理视图
- **拖拽导入**：支持从系统文件管理器拖拽文件/文件夹到子项区域，自动导入
  - 拖拽悬停时区域高亮
  - 释放后显示导入进度

## 4. 中间对话区

### 4.1 消息流区域

#### 4.1.1 消息气泡布局

```
┌───────────────────────────────────────────────┐
│                                               │
│                        ┌─────────────────────┐│
│                        │ 用户消息（右对齐）     ││  ← 靠右，主色背景
│                        │ 帮我找一下量子计算    ││
│                        │ 最新的综述文章        ││
│                        └─────────────────────┘│
│                                               │
│ ┌────────────────────────────────────┐        │
│ │ AI回复（左对齐）                     │        │  ← 靠左，白色卡片背景
│ │                                     │        │
│ │ 好的，我来帮你检索...                │        │
│ │                                     │        │
│ │ ┌ 工具调用 ─────────────────────┐   │        │
│ │ │ search_papers(keywords=...)   │   │        │  ← 可展开/折叠
│ │ │ 用时: 1.2s | 结果: 15条  [▾]  │   │        │
│ │ └──────────────────────────────┘   │        │
│ └────────────────────────────────────┘        │
│                                               │
└───────────────────────────────────────────────┘
```

#### 4.1.2 消息类型渲染规范

| 消息类型 | 渲染方式 | 交互 |
|---------|---------|------|
| **纯文本** | 支持 Markdown 渲染（标题/列表/加粗/斜体/链接） | — |
| **代码块** | 暗色背景 `#1E1E2E`，语法高亮（Prism.js），行号 | 右上角「复制」按钮，点击复制全文 |
| **表格** | Markdown 表格渲染，斑马纹 | — |
| **图片** | 缩略图（最大宽 400px），点击放大 | 点击后在右侧预览Tab全屏查看 |
| **文件卡片** | 文件类型图标 + 文件名 + 大小 + 下载按钮 | 点击在右侧预览Tab打开；PDF/图片直接预览 |
| **工具调用卡片** | 折叠状态：工具名 + 耗时 + 结果数；展开状态：完整参数 + 返回摘要 | 点击标题栏折叠/展开 |
| **Mermaid 图表** | 实时渲染为 SVG | — |

#### 4.1.3 工具调用卡片详细规格

**折叠状态**：
```
┌─────────────────────────────────────────────────────┐
│ 🔧 search_papers  ·  用时 1.2s  ·  返回 15 条    [▸] │
└─────────────────────────────────────────────────────┘
```

**展开状态**：
```
┌─────────────────────────────────────────────────────┐
│ 🔧 search_papers  ·  用时 1.2s  ·  返回 15 条    [▾] │
├─────────────────────────────────────────────────────┤
│ 参数:                                                │
│   keywords: "quantum computing survey"               │
│   year_from: 2023                                    │
│   year_to: 2026                                      │
│   limit: 20                                          │
├─────────────────────────────────────────────────────┤
│ 结果摘要:                                             │
│   1. A Survey on Quantum... (Nature, 2025)           │
│   2. Recent Advances in... (Science, 2024)           │
│   ... 共 15 条                                       │
└─────────────────────────────────────────────────────┘
```

#### 4.1.4 欢迎引导页（无对话时）

空状态时显示：
- 产品 Logo（居中，大尺寸）
- 欢迎语："你好，我是 SciAgent，你的科研全流程助手"
- 快捷引导卡片（3~4 张）："开始文献检索" / "帮我写综述" / "创建仿真实验" / "管理我的知识库"

### 4.2 底部输入区

#### 4.2.1 布局

```
┌─────────────────────────────────────────────────────┐
│ ┌─────────────────────────────────────────────────┐ │
│ │ [已选文件缩略图1] [×]  [已选文件缩略图2] [×]     │ │  ← 有文件时显示
│ └─────────────────────────────────────────────────┘ │
│ ┌─────────────────────────────────────────────────┐ │
│ │ 多行输入框                                      │ │
│ │                                                 │ │
│ │                                                 │ │
│ └─────────────────────────────────────────────────┘ │
│ ┌───────────┬──────────┬──────────┬────────┬────┐ │
│ │ [技能 ▾]  │ [MCP ▾]  │ [连接器▾] │ [📎添加] │ → │ │
│ └───────────┴──────────┴──────────┴────────┴────┘ │
└─────────────────────────────────────────────────────┘
```

#### 4.2.2 输入框

- **最小行数**：2 行
- **最大行数**：8 行（超出后内部滚动）
- **自动伸缩**：随内容增加行数，达到最大行数后停止增长
- **Markdown 快捷语法**：支持 `**加粗**`、`` `代码` ``、`> 引用` 等（实时预览可选，通过设置开关）
- **键盘快捷键**：
  - `Enter` — 发送消息
  - `Shift+Enter` — 换行
  - `Ctrl+Enter` — 强制发送（即使输入框逻辑判定不应发送）

#### 4.2.3 技能下拉菜单

- **触发**：点击 `[技能 ▾]` 按钮
- **菜单规格**：宽度 320px，最大高度 400px
- **内容结构**：

┌──────────────────────────────────┐

├──────────────────────────────────┤

│   └ 参考文献格式化               │
├──────────────────────────────────┤
│ ✏️ 写作辅助                      │
│   ├ 学术润色                     │
│   ├ 语句降重                     │
│   └ 格式匹配                     │
├──────────────────────────────────┤
│ 🔬 实验仿真                      │
│   ├ Python 代码执行              │
│   └ 数据分析                     │
├──────────────────────────────────┤
│ 📊 数据可视化                    │
│   ├ 生成图表                     │
│   └ 统计报表                     │
└──────────────────────────────────┘
```

- **技能来源**：系统内置 + 工具广场已安装
- **点击行为**：在输入框中插入技能调用标记（如 `/skill:literature_review`），光标定位到参数区
- **关闭方式**：点击外部 / `Esc` / 选择技能后关闭

#### 4.2.4 MCP 下拉菜单

- **触发**：点击 `[MCP ▾]` 按钮
- **菜单规格**：宽度 280px，最大高度 360px
- **内容结构**：

```
┌──────────────────────────────────┐
│ MCP 连接器                       │
├──────────────────────────────────┤
│ 🟢 科应 API (keying)             │  ← 状态指示灯
│    已连接  · 配额 850/1000       │
│    [禁用]                        │  ← 切换按钮
├──────────────────────────────────┤
│ 🔴 GitHub                        │
│    未连接                        │
│    [启用]                        │
├──────────────────────────────────┤
│ 🟢 Zotero                        │
│    已连接                        │
│    [禁用]                        │
├──────────────────────────────────┤
│ ⚪ 自定义 MCP                     │
│    待配置                        │
│    [配置]                        │
├──────────────────────────────────┤
│ ──────────────────────────────── │
│  [+ 添加连接器]                  │  ← 入口
└──────────────────────────────────┘
```

- **状态指示灯**：
  - 🟢 绿色：已连接，正常
  - 🔴 红色：已配置但连接失败
  - ⚪ 灰色：未配置

## 6. 拖拽与折叠交互

### 6.1 分隔条拖拽

#### 6.1.1 规格参数

| 参数 | 左侧分隔条 | 右侧分隔条 |
|------|-----------|-----------|
| 默认宽度 | 4px | 4px |
| hover 宽度 | 8px | 8px |
| hover 颜色 | `#D0C8E8`（半透明主色） | 同左 |
| 光标 | `col-resize` | `col-resize` |
| 可拖拽范围 | 180px ~ 360px | 280px ~ 600px |
| 拖拽反馈 | 实时跟随鼠标，半透明虚线引导线 | 同左 |

#### 6.1.2 拖拽行为

1. **mousedown**：分隔条上按下鼠标 → 记录起始 X 坐标和当前栏宽
2. **mousemove**：计算偏移量 `deltaX`，更新栏宽（实时渲染）
3. **mouseup**：释放 → 栏宽取整到偶数像素 → 持久化到 `localStorage`
4. **边界处理**：达到最小/最大宽度时停止跟随，分隔条颜色变红（200ms）提示

#### 6.1.3 持久化键名

| 键名 | 值 | 说明 |
|------|---|------|
| `sciagent.layout.left_width` | `240` | 左侧栏宽度（px） |
| `sciagent.layout.right_width` | `400` | 右侧栏宽度（px） |
| `sciagent.layout.left_collapsed` | `false` | 左侧栏是否折叠 |
| `sciagent.layout.right_collapsed` | `false` | 右侧栏是否折叠 |

### 6.2 折叠按钮

#### 6.2.1 位置与样式

- **左侧栏折叠按钮**：左侧栏右上角，`◀` 图标，hover 显示 Tooltip "折叠侧边栏"
- **右侧工作区折叠按钮**：右侧工作区 Tab 栏最右端，`▶` 图标
- **展开按钮**：
  - 左侧栏折叠后：在界面左侧边缘显示竖条 `▶`，点击展开
  - 右侧工作区折叠后：在界面右侧边缘显示竖条 `◀`，点击展开

#### 6.2.2 折叠动画

- **持续时间**：250ms
- **缓动函数**：`cubic-bezier(0.4, 0, 0.2, 1)`（Material Design 标准缓动）
- **效果**：栏宽过渡到 0，内容 `opacity` 渐变消失

## 8. 响应式规则

### 8.1 断点定义

| 断点 | 条件 | 行为 |
|------|------|------|
| `>= 1200px` | 桌面宽屏 | 三栏完整显示，默认宽度生效 |
| `900px ~ 1199px` | 桌面窄屏 | 右侧工作区自动折叠（可手动展开） |
| `600px ~ 899px` | 平板 | 左侧栏自动折叠为图标模式（可手动展开） |
| `< 600px` | 手机 | 左侧栏图标模式，右侧工作区隐藏，对话区全宽 |

### 8.2 图标模式（左侧栏）

│ 💬 │  ← 会话历史
│    │
│ ⏰ │  ← 自动任务

│ 🔬 │  ← 科研全流程
