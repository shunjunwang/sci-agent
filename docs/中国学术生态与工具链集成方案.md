## 1. 中国学术数据库对接方案

### 1.1 现状与差距分析

现有 SPEC 中 `DataSourcePlugin` 仅实现了 OpenAlex 适配器（`OpenAlexAdapter`），覆盖英文文献源。中国学术数据库的缺失导致：

| 差距项 | 影响 | 严重度 |
|--------|------|--------|
| 中文文献无法检索 | 用户无法找到知网/万方收录的中文论文 | 🔴 高 |
| 引用格式不兼容 | GB/T 7714 标准无法自动生成 | 🔴 高 |
| 文献模型不区分语种 | `papers` 表无 `language` 字段，检索结果混乱 | 🟡 中 |
| 无中文期刊模板 | 无法生成符合国内期刊投稿要求的文献列表 | 🟡 中 |

#### 1.2.1 CNKI（知网）

**现状**：无公开 API。**策略：合规元数据索引。**

| 维度 | 方案 |
|------|------|
| 数据来源 | 知网公开的期刊/论文元数据页面（标题、作者、摘要、关键词、DOI） |
| 采集方式 | 利用 CrossRef / DOI 注册中心公开元数据，通过 DOI 反向映射知网文献 |
| 限制 | **绝不采集全文**，仅索引公开元数据字段 |
| 合规依据 | 元数据不受著作权法保护；仅索引不存储原文 |
| 更新频率 | 每日增量更新（新增收录的 DOI 条目） |

**实现要点**：

```python
# cnki_adapter.py — 核心结构
class CNKIAdapter(DataSourcePlugin):
    """知网适配器 — 仅基于公开元数据索引"""

    source_name: str = "cnki"
    priority: int = 80            # 中文场景优先级最高
    language: str = "zh"

    # ---- 检索入口 ----
    async def search(
        self,
        query: SearchQuery,
        limit: int = 20,
    ) -> List[PaperDTO]:
        """
        检索策略：
        1. 先查本地 metadata_index（已从 DOI 注册中心同步的知网元数据）
        2. 补充查询 Crossref API（带 CNKI 前缀过滤）
        3. 返回去重后的统一 PaperDTO 列表
        """
        ...

    # ---- 本地元数据索引维护 ----
    async def sync_metadata_index(self):
        """增量同步公开元数据到本地 SQLite/Postgres 索引"""
        ...

    def _parse_cnki_metadata(self, raw: dict) -> PaperDTO:
        """将知网元数据映射到统一 PaperDTO"""
        return PaperDTO(
            title=raw.get("title"),
            authors=raw.get("authors", []),
            abstract=raw.get("abstract"),
            doi=raw.get("doi"),
            source_db="cnki",
            language="zh",
            ...
        )
```

#### 1.2.2 万方数据 (Wanfang)

| 维度 | 方案 |
|------|------|
| API 状态 | 万方提供机构 API（需订阅），个人开发者可申请试用 |
| 对接方式 | 封装 `WanfangAdapter`，优先走 API；API 不可用则降级到公开元数据 |
| 检索参数 | 支持中文关键词 / 作者 / 机构 / 基金项目 |
| 返回字段 | 标题、作者、摘要、关键词、期刊、年卷期、DOI |

```python
class WanfangAdapter(DataSourcePlugin):
    source_name: str = "wanfang"
    priority: int = 70

    async def search(self, query, limit=20):
        if self._api_available():
            return await self._call_api(query, limit)
        else:
            return await self._fallback_metadata_search(query, limit)
```

**路由规则**：

```python
class QueryRouter:
    """根据检索意图路由到合适的数据库适配器"""

    CNKI_KEYWORDS = re.compile(r'[\u4e00-\u9fff]{3,}')  # 含≥3个连续中文

    def route(self, query: str) -> List[DataSourcePlugin]:
        adapters: List[DataSourcePlugin] = []

        # 中文查询 → 优先 CNKI + 万方 + 维普 + 百度学术
        if self.CNKI_KEYWORDS.search(query):
            adapters.extend([
                CNKIAdapter(),
                WanfangAdapter(),
                CQVIPAdapter(),
                BaiduScholarAdapter(),
            ])

        # 始终并行查询 OpenAlex（覆盖中英文文献）
        adapters.append(OpenAlexAdapter())

        return sorted(adapters, key=lambda a: a.priority, reverse=True)
```

### 2.2 ImportService 抽象层设计

```python
from abc import ABC, abstractmethod
from enum import Enum
from dataclasses import dataclass, field
from typing import List, Optional, AsyncIterator

# ============================================================
# 公共类型
# ============================================================

class ImportSource(Enum):
    ZOTERO    = "zotero"
    MENDELEY  = "mendeley"
    ENDNOTE   = "endnote"

class SyncDirection(Enum):
    PULL_ONLY   = "pull"      # 仅从管理器拉取
    PUSH_ONLY   = "push"      # 仅推送到管理器
    BIDIRECTIONAL = "bidirectional"

@dataclass
class ImportResult:
    source: ImportSource
    total_fetched: int
    new_added: int
    duplicates_skipped: int
    errors: List[str] = field(default_factory=list)

@dataclass
class SyncStatus:
    source: ImportSource
    last_sync_at: Optional[str]
    items_synced: int
    direction: SyncDirection

# ============================================================
# 抽象导入器
# ============================================================

class BaseImporter(ABC):
    """所有参考文献管理器导入器的基类"""

    source: ImportSource

    @abstractmethod
    async def authorize(self, credentials: dict) -> str:
        """执行 OAuth 授权流程，返回 access_token"""
        ...

    @abstractmethod
    async def fetch_library(
        self,
        token: str,
        since: Optional[str] = None,   # 增量同步时间戳
        limit: int = 500,
    ) -> AsyncIterator[List[PaperDTO]]:
        """分页拉取文献库"""
        ...

    @abstractmethod
    async def push_items(
        self,
        token: str,
        papers: List[PaperDTO],
    ) -> List[str]:
        """将文献推送回管理器，返回远程 ID 列表"""
        ...

    async def import_to_user_library(
        self,
        token: str,
        dedup_engine: "DedupEngine",
    ) -> ImportResult:
        """标准导入流程：拉取 → 去重 → 写入 user_library"""
        result = ImportResult(source=self.source, total_fetched=0,
                              new_added=0, duplicates_skipped=0)

        async for batch in self.fetch_library(token):
            for paper in batch:
                result.total_fetched += 1
                if dedup_engine.is_duplicate(paper):
                    result.duplicates_skipped += 1
                else:
                    await user_library.insert(paper)
                    result.new_added += 1

        return result

# ============================================================
# ImportService 注册中心
# ============================================================

class ImportService:
    """插件式导入服务注册中心"""

    _importers: dict[ImportSource, BaseImporter] = {}

    @classmethod
    def register(cls, importer: BaseImporter):
        cls._importers[importer.source] = importer

    @classmethod
    def get(cls, source: ImportSource) -> BaseImporter:
        if source not in cls._importers:
            raise ValueError(f"No importer registered for {source}")
        return cls._importers[source]

    @classmethod
    def list_sources(cls) -> List[ImportSource]:
        return list(cls._importers.keys())
```

### 2.3 Zotero 集成

**双向同步**是 Zotero 集成的核心价值：用户在 Zotero 中整理文献 → sci-agent 读入分析 → 标注结果回写 Zotero。

```
┌───────────────────────────────────────────────────────────────────┐
│                     Zotero 双向同步流程                             │
├───────────────────────────────────────────────────────────────────┤
│                                                                    │
│   用户浏览器                       sci-agent                       │
│   ┌─────────┐                    ┌──────────────┐                  │
│   │ 点击授权 │ ──────────────────▶│ 生成 OAuth   │                  │
│   │         │◀───────────────────│ 授权 URL     │                  │
│   │ 确认授权 │ ──────────────────▶│ 回调获取     │                  │
│   │         │                    │ access_token │                  │
│   └─────────┘                    └──────┬───────┘                  │
│                                         │                          │
│                                         ▼                          │
│                    ┌──────────────────────────────┐                │
│                    │  Zotero Web API               │                │
│                    │  GET /users/{id}/items        │                │
│                    │  ← 拉取文献库（含附件/笔记）    │                │
│                    └──────────────┬───────────────┘                │
│                                   │                                │
│                                   ▼                                │
│                    ┌──────────────────────────────┐                │
│                    │  sci-agent 处理                │                │
│                    │  • 去重合并                    │                │
│                    │  • 写入 user_library           │                │
│                    │  • AI 分析 / 标注             │                │
│                    └──────────────┬───────────────┘                │
│                                   │                                │
│                                   ▼                                │
│                    ┌──────────────────────────────┐                │
│                    │  Zotero Web API               │                │
│                    │  PUT /users/{id}/items/{key}  │                │
│                    │  ← 回写标注/标签到 Zotero      │                │
│                    └──────────────────────────────┘                │
│                                                                    │
└───────────────────────────────────────────────────────────────────┘
```

```python
class ZoteroImporter(BaseImporter):
    """Zotero Web API 集成实现"""

    source = ImportSource.ZOTERO
    API_BASE = "https://api.zotero.org"

    ZOTERO_TO_PAPER_MAP = {
        # Zotero itemType → 内部处理方式
        "journalArticle": "journal_article",
        "conferencePaper": "conference_paper",
        "book": "book",
        "bookSection": "book_chapter",
        "thesis": "thesis",
        "preprint": "preprint",
        "report": "report",
    }

    async def authorize(self, credentials: dict) -> str:
        """
        Zotero OAuth 流程：
        1. 用户在 Zotero 开发者页面创建应用，获取 client_key + client_secret
        2. 引导用户访问授权 URL
        3. 回调获取 oauth_token + oauth_token_secret
        4. 返回组合凭证
        """
        ...

    async def fetch_library(
        self, token: str, since: Optional[str] = None, limit=500
    ) -> AsyncIterator[List[PaperDTO]]:
        """
        调用 Zotero Web API v3：
        GET /users/{userID}/items?limit={limit}&since={since}
        """
        headers = {"Authorization": f"Bearer {token}",
                    "Zotero-API-Version": "3"}
        start = 0
        while True:
            url = f"{self.API_BASE}/users/{self._user_id}/items"
            params = {"limit": limit, "start": start}
            if since:
                params["since"] = since  # Zotero 版本号，实现增量同步

            resp = await self._http_get(url, headers, params)
            items = self._parse_items(resp.json())
            if not items:
                break
            yield [self._to_paper_dto(item) for item in items
                   if self._is_academic(item)]
            start += limit

    async def push_items(self, token, papers) -> List[str]:
        """
        将标注结果回写到 Zotero：
        PUT /users/{userID}/items/{itemKey}
        仅更新 tags 和 notes 字段，不改动原文元数据
        """
        remote_ids = []
        for paper in papers:
            zotero_key = self._resolve_key(paper)  # 通过 DOI 反查 Zotero key
            if zotero_key:
                payload = {
                    "tags": [{"tag": t} for t in paper.tags],
                    "note": paper.user_annotation,
                }
                url = f"{self.API_BASE}/users/{self._user_id}/items/{zotero_key}"
                await self._http_put(url, headers, payload)
                remote_ids.append(zotero_key)
        return remote_ids

    def _to_paper_dto(self, zotero_item: dict) -> PaperDTO:
        data = zotero_item.get("data", {})
        return PaperDTO(
            title=data.get("title", ""),
            authors=self._parse_creators(data.get("creators", [])),
            abstract=data.get("abstractNote", ""),
            doi=data.get("DOI"),
```

```python
class RISParser:
    """RIS 格式解析器 — 支持 EndNote / Zotero 导出的 .ris 文件"""

    TAG_MAP = {
        "TY": "type",
        "TI": "title",
        "T1": "title",
        "AU": "author",
        "A1": "author",
        "AB": "abstract",
        "N2": "abstract",
        "DO": "doi",
        "PY": "year",
        "Y1": "year",
        "JO": "journal",
        "JF": "journal",
        "JA": "journal_abbrev",
        "VL": "volume",
        "IS": "issue",
        "SP": "start_page",
        "EP": "end_page",
        "KW": "keywords",
        "UR": "url",
        "L1": "file_attachment",
        "SN": "issn",
        "LA": "language",
    }

    def parse(self, file_path: str) -> List[PaperDTO]:
        papers = []
        current_record: dict = {}
        current_tag = None

        with open(file_path, "r", encoding="utf-8") as f:
            for line in f:
                line = line.rstrip("\n\r")
                if line.startswith("ER  -") or line == "":
                    if current_record:
                        papers.append(self._build_paper(current_record))
                        current_record = {}
                    continue

                match = re.match(r"^(\w{2})\s{2}-\s(.*)", line)
                if match:
                    tag, value = match.groups()
                    current_tag = self.TAG_MAP.get(tag, tag.lower())
                    current_record.setdefault(current_tag, []).append(value)
                elif current_tag and line.startswith("   "):
                    # 续行
                    current_record[current_tag][-1] += " " + line.strip()

        return papers
```

```python
class DedupEngine:
    """文献去重引擎"""

    async def is_duplicate(self, paper: PaperDTO) -> bool:
        # 1. DOI 精确匹配（最高优先级）
        if paper.doi:
            existing = await self._find_by_doi(paper.doi)
            if existing:
                await self._merge_if_needed(existing, paper)
                return True

        # 2. 标题模糊匹配（去掉标点和空格后计算相似度）
        normalized_title = self._normalize(paper.title)
        candidates = await self._find_by_title_prefix(normalized_title[:50])
        for c in candidates:
            if self._levenshtein_ratio(normalized_title,
                                       self._normalize(c.title)) > 0.90:
                await self._merge_if_needed(c, paper)
                return True

        return False

    @staticmethod
    def _normalize(s: str) -> str:
        return re.sub(r'[^\w]', '', s.lower())
```

### 3.1 GB/T 7714 格式化引擎

GB/T 7714-2015 是中国参考文献著录国家标准，几乎所有的中文期刊、学位论文都要求使用此格式。

```
┌─────────────────────────────────────────────────────────────┐
│                GB/T 7714 Formatter Pipeline                  │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  PaperDTO  ──▶  TypeRouter  ──▶  Formatter  ──▶  输出       │
│                  (判断文献类型)   (按规则渲染)               │
│                                                              │
│  支持类型：                                                   │
│  [J] 期刊论文   [M] 专著        [C] 会议论文集               │
│  [D] 学位论文   [R] 报告        [P] 专利                     │
│  [S] 标准       [EB/OL] 电子资源  [N] 报纸                   │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

```python
from enum import Enum

class GBT7714Type(Enum):
    JOURNAL    = ("J", "期刊论文")
    MONOGRAPH  = ("M", "专著")
    PROCEEDING = ("C", "会议论文集")
    DISSERTATION = ("D", "学位论文")
    REPORT     = ("R", "报告")
    PATENT     = ("P", "专利")
    STANDARD   = ("S", "标准")
    ELECTRONIC = ("EB/OL", "电子资源")
    NEWSPAPER  = ("N", "报纸")

class GBT7714Formatter:
    """GB/T 7714-2015 参考文献格式化器"""

    def format(self, paper: PaperDTO, gbt_type: GBT7714Type) -> str:
        handler = {
            GBT7714Type.JOURNAL:    self._format_journal,
            GBT7714Type.MONOGRAPH:  self._format_monograph,
            GBT7714Type.PROCEEDING: self._format_proceeding,
            GBT7714Type.DISSERTATION: self._format_dissertation,
            GBT7714Type.ELECTRONIC: self._format_electronic,
        }.get(gbt_type, self._format_generic)

        return handler(paper)

    def _format_journal(self, p: PaperDTO) -> str:
        """
        格式：[序号] 作者. 题名[J]. 刊名, 出版年, 卷号(期号): 起止页码.

        示例：
        [1] 张三, 李四. 基于深度学习的文本分类研究[J].
            计算机学报, 2023, 46(3): 521-538.
        """
        authors = self._format_authors(p.authors)
        title = p.cn_title or p.title
        journal = p.cn_journal or p.journal or ""
        year = p.year or ""
        volume = p.volume or ""
        issue = p.issue or ""
        pages = p.pages or ""

        citation = f"{authors}. {title}[J]."
        if journal:
            citation += f" {journal},"
        if year:
            citation += f" {year}"
        if volume:
            citation += f", {volume}"
            if issue:
                citation += f"({issue})"
        if pages:
            citation += f": {pages}"
        citation += "."

        return citation

    def _format_dissertation(self, p: PaperDTO) -> str:
        """
        格式：[序号] 作者. 题名[D]. 出版地: 学位授予单位, 出版年.

        示例：
        [2] 王五. 基于注意力机制的图像分割方法研究[D].
            北京: 清华大学, 2022.
        """
        authors = self._format_authors(p.authors)
        title = p.cn_title or p.title
        institution = p.publisher or p.institution or ""
        year = p.year or ""
        location = p.pub_location or ""

        citation = f"{authors}. {title}[D]."
        if location and institution:
            citation += f" {location}: {institution},"
        elif institution:
            citation += f" {institution},"
        if year:
            citation += f" {year}."

        return citation

    def _format_authors(self, authors: List[AuthorDTO]) -> str:
        """作者格式化：3人以下全列，超过3人取前3人加', 等'"""
        names = [a.cn_name or a.name for a in authors]
        if len(names) <= 3:
            return ", ".join(names)
        return ", ".join(names[:3]) + ", 等"

    def format_list(
        self, papers: List[PaperDTO],
        numbered: bool = True
    ) -> str:
        """批量格式化，自动编号"""
        lines = []
        for i, paper in enumerate(papers):
            gbt_type = self._infer_type(paper)
            citation = self.format(paper, gbt_type)
            if numbered:
                lines.append(f"[{i+1}] {citation}")
            else:
                lines.append(citation)
        return "\n".join(lines)
```

### 3.3 BibTeX 中文兼容处理

**问题**：标准 BibTeX 对中文字符支持有限，需要特殊处理作者拼音、中文姓名的分拆。

```python
@dataclass
class AuthorDTO:
    name: str               # 英文名，如 "San Zhang"
    cn_name: Optional[str]  # 中文名，如 "张三"
    pinyin: Optional[str]   # 拼音全名，如 "Zhang San"
    pinyin_surname: Optional[str]   # 拼音姓，如 "Zhang"
    pinyin_givenname: Optional[str] # 拼音名，如 "San"

class BibTeXCNFormatter:
    """BibTeX 中文兼容格式化器"""

    def format(self, paper: PaperDTO, cite_key: str) -> str:
        entry_type = self._map_bibtex_type(paper)

        fields = []
        # 作者处理
        fields.append(self._format_author_field(paper.authors))
        # 标题：中文标题放 title，英文标题放 booktitle/journal
        fields.append(self._format_title_field(paper))
        # 其它字段
        if paper.journal:
            fields.append(f"  journal = {{{paper.journal}}}")
        if paper.year:
            fields.append(f"  year = {{{paper.year}}}")
        if paper.volume:
            fields.append(f"  volume = {{{paper.volume}}}")
        if paper.pages:
            fields.append(f"  pages = {{{paper.pages}}}")
        if paper.doi:
            fields.append(f"  doi = {{{paper.doi}}}")

        # 中文特有 — language 字段
        if paper.language == "zh":
            fields.append(f"  language = {{Chinese}}")

        body = ",\n".join(fields)
        return f"@{entry_type}{{{cite_key},\n{body}\n}}"

    def _format_author_field(self, authors: List[AuthorDTO]) -> str:
        """
        中文作者 BibTeX 处理策略：

        1. 有拼音 → 使用 {Surname}, {GivenName} 格式
           author = {Zhang, San and Li, Si}

        2. 仅有中文名 → 使用全名并用 {} 保护
           author = {{张三} and {李四}}

        3. 中英混合 → 分开处理
           author = {Smith, John and {王五}}
        """
        parts = []
        for a in authors:
            if a.pinyin_surname and a.pinyin_givenname:
                parts.append(f"{a.pinyin_surname}, {a.pinyin_givenname}")
            elif a.cn_name:
                parts.append(f"{{{a.cn_name}}}")
            else:
                parts.append(a.name)
        return f"  author = {{{' and '.join(parts)}}}"

    def _format_title_field(self, paper: PaperDTO) -> str:
        """中文文献标题处理"""
        if paper.language == "zh" and paper.cn_title:
            # 中文标题加 {} 保护，防止 bibtex 自动大小写转换
            return f"  title = {{{{{paper.cn_title}}}}}"
        return f"  title = {{{paper.title}}}"

    def _map_bibtex_type(self, paper: PaperDTO) -> str:
        """PaperDTO → BibTeX entry type"""
        type_map = {
            "journal_article": "article",
            "conference_paper": "inproceedings",
            "book": "book",
            "book_chapter": "incollection",
            "thesis": "phdthesis",
            "report": "techreport",
        }
        return type_map.get(paper.pub_type, "misc")
```
